Features

Everything you need to ship

BassEngine gives you a complete toolkit out of the box — no plugins needed for the core gameplay loop.

⚡

Fast Renderer

WebGL-powered 2D/3D renderer with batched draw calls, sprite atlases, and shader support baked in.

🎮

Input System

Unified handling for keyboard, mouse, gamepad, and touch with configurable action maps and dead zones.

🔊

Audio Engine

Web Audio API integration with 3D positional audio, reverb chains, and real-time mixer out of the box.

🧩

ECS Architecture

Entity-Component-System design keeps your code decoupled, testable, and orders-of-magnitude faster.

🌎

Scene Management

Tree-based scene graph with async asset loading, animated transitions, and persistent world state.

🔧

Built-in Physics

Lightweight AABB and circle collision, rigidbody simulation, and raycasting — no heavy deps required.

Quickstart

Up and running
in minutes

Three steps from zero to a running game loop with renderer, input, and physics.

1

Install BassEngine

Run npm i bass-engine or drop in the CDN script tag.

2

Create a Scene

Extend Scene and compose your world from entities and components.

3

Call BassEngine.run()

Your game loop, renderer, and physics all start automatically.

main.js 
// Import what you need
import {
  BassEngine, Scene,
  Sprite, Input
} from 'bass-engine';

class GameScene extends Scene {
  onCreate() {
    this.player = new Sprite({
      texture: 'player.png',
      x: 100, y: 200
    });
    this.add(this.player);
  }

  onUpdate(dt) {
    if (Input.isDown('ArrowRight'))
      this.player.x += 200 * dt;
  }
}

BassEngine.run(new GameScene());
