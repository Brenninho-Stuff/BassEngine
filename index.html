Features

Everything you need to ship

BassEngine gives you a full toolkit — no plugins required for the core gameplay loop.

⚡

Fast Renderer

WebGL-powered 2D/3D renderer with batched draw calls, sprite atlases, and shader support out of the box.

🎮

Input System

Unified input handling for keyboard, mouse, gamepad, and touch with action mapping and dead zones.

🔊

Audio Engine

Web Audio API-based engine with 3D positional audio, reverb, and real-time mixing built in.

🧩

ECS Architecture

Entity-Component-System design keeps your game code decoupled, testable, and blazing fast.

🌎

Scene Management

Tree-based scene graph with async loading, transitions, and persistent global state.

🔧

Built-in Physics

Lightweight AABB and circle collision, rigidbody physics, and raycasting without heavy dependencies.

Quickstart

Up and running in minutes

Three steps and you have a working game loop, renderer, and input system.

1

Install BassEngine

Add it to your project via npm or include the CDN script tag.

2

Create a Scene

Extend the base Scene class and register your entities.

3

Run the Engine

Call BassEngine.run() and your game loop starts instantly.

// main.js
import { BassEngine, Scene, Sprite } from 'bass-engine';

class GameScene extends Scene {
  onCreate() {
    this.player = new Sprite({
      texture: 'player.png',
      x: 100, y: 200,
    });
    this.add(this.player);
  }

  onUpdate(dt) {
    if (Input.isDown('ArrowRight')) {
      this.player.x += 200 * dt;
    }
  }
}

BassEngine.run(new GameScene());
